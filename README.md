
# Customer Churn Prediction using Artificial Neural Networks

This project predicts whether a bank customer is likely to leave (churn)
the bank using an Artificial Neural Network (ANN) built with
TensorFlow/Keras.

## Project Overview

Customer churn prediction is an important problem in the banking
industry. By analyzing customer features such as credit score,
geography, gender, age, balance, and salary, the model predicts whether
a customer will exit the bank.

## Technologies Used

* Python
* Pandas
* NumPy
* TensorFlow / Keras
* Scikit-learn
* Matplotlib

## Workflow

1. Data preprocessing
2. One-hot encoding of categorical variables
3. Feature scaling using StandardScaler
4. Training an Artificial Neural Network
5. Model evaluation using accuracy and other metrics

## Dataset

The dataset contains bank customer information such as: - Credit Score -
Geography - Gender - Age - Balance - Estimated Salary - Churn (target
variable)

## Model

An Artificial Neural Network was trained using TensorFlow/Keras with
binary cross-entropy loss and Adam optimizer to classify whether a
customer will churn.

## Future Improvements

* Add more evaluation metrics such as Precision, Recall, and ROC-AUC
* Deploy the model using Streamlit for interactive predictions
* Compare ANN performance with other ML models
